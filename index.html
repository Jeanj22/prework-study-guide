Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport
  • The base element defines the base URL for a webpage.
  • The style element defines the CSS styles associated with a website.
  • The script element embeds client-side scripts in the HTML document.
  • The p element represents a paragraph or block of text.
  • The a element (which stands for anchor) creates links to the same webpage or other webpages.
  • The href attribute points to the URL
  • The src attribute is the most important because it defines the location of the image
  • The img element contains information about images that are displayed on the webpage.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The header appears at the top of the page and usually contains a navigation menu and/or introductory content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The most common way to style a webpage is by using an external CSS style sheet.
  • When using an external style sheet, we link the file using a link element placed in the HTML head.
  • A link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • The period preceding card designates this as a class selector.
  • The border property assigns a border
  • The box-shadow CSS property adds shadow effects around an element's frame.
  • Assigning colours can be done by using semantic references such as gray or blue or by using its hexadecimal code, or hex #.
